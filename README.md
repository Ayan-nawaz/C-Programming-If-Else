# C Programming If-Else

A collection of C programs based on **if**, **if-else**, **nested if-else**, and **else-if** statements. This repository is designed for beginners to strengthen decision-making and problem-solving skills in C.

---

## 📚 Programs Included

- Absolute Number
- Divisible by 5 or 3
- Else Statement
- If-Else Statement
- Selling Price & Cost Price
- More programs will be added regularly...

- Simple `if` Statement
- `if-else` Statement
- Nested `if-else`
- `else-if` Ladder
- Decision Making
- Relational Operators
- Logical Operators
- Basic Problem Solving

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Ayan-nawaz/C-Programming-If-Else.git
   ```

2. Open any `.c` file.

3. Compile the program:
   ```bash
   gcc filename.c -o output
   ```

4. Run the executable:
   ```bash
   ./output
   ```
